# Requirements: @Bangkok Dental Clinic

**Defined:** 2025-02-01
**Core Value:** Communicate Dr. Photjana's world-class credentials and 40+ years of expertise through understated elegance

## v1 Requirements

Requirements for initial release. Each maps to roadmap phases.

### Hero

- [ ] **HERO-01**: Full-viewport hero with "Smiles Perfected" headline
- [ ] **HERO-02**: Vanta.js Waves animated background (light purple tones)
- [ ] **HERO-03**: Subtitle with clinic positioning
- [ ] **HERO-04**: Primary CTA button "Book Appointment"

### Navigation

- [ ] **NAV-01**: Minimal fixed navigation with logo
- [ ] **NAV-02**: Smooth scroll to sections
- [ ] **NAV-03**: Mobile hamburger menu

### Credentials

- [ ] **CRED-01**: Dr. Photjana profile section
- [ ] **CRED-02**: Display all degrees (BSc, DDS, MDSc, DScD)
- [ ] **CRED-03**: University affiliations (Sydney, Boston, Temple, Mahidol)
- [ ] **CRED-04**: 40+ years experience highlight

### Services

- [ ] **SERV-01**: Orthodontics/braces services section
- [ ] **SERV-02**: Grid layout with breathing room
- [ ] **SERV-03**: SVG icons for each service

### Social Proof

- [ ] **SOCIAL-01**: Customer photos carousel
- [ ] **SOCIAL-02**: 12 placeholder image slots
- [ ] **SOCIAL-03**: Smooth slide/swipe interaction

### Location

- [ ] **LOC-01**: Embedded Google Maps
- [ ] **LOC-02**: Address displayed in Thai and English
- [ ] **LOC-03**: Phone number with click-to-call

### Booking

- [ ] **BOOK-01**: Appointment booking section
- [ ] **BOOK-02**: Contact form or calendar embed
- [ ] **BOOK-03**: Clear CTA

### Design System

- [ ] **DESIGN-01**: Light purple + soft gray palette
- [ ] **DESIGN-02**: Spectral + Lato typography
- [ ] **DESIGN-03**: Massive whitespace throughout
- [ ] **DESIGN-04**: SVG icons only (no emojis)
- [ ] **DESIGN-05**: Smooth scroll animations
- [ ] **DESIGN-06**: Mobile responsive (all breakpoints)

## Out of Scope

| Feature | Reason |
|---------|--------|
| Pricing display | Clinic prefers consultation-based pricing |
| Patient portal/login | Not needed for marketing site |
| Blog/news section | Not needed for marketing focus |
| Multiple pages | Single landing page sufficient for v1 |
| bookd-win-win integration | Separate project entirely |
